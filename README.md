# Modelling malaria routine surveillance data to inform seasonal malaria chemoprevention strategy in Moissala, Southern Chad.
The code in this repository reproduces the results and plots in the paper: 

*Modelling malaria routine surveillance data to inform seasonal malaria chemoprevention strategy in Moissala, Southern Chad.* **Nicholas Putney, Jessica Sayyad-Hilario, Israel Ukawuba1, Francesco Grandesso3, Saschveen Singh, Prince Djuma Safari, Emilie Pothin, Beatrice Filippini, Elkoussing Djovouna, Mahamat Saleh Issakha Diar, Clara Champagne, Anton Camacho** (not yet published).

Running the code requires the *malclimsim* R-package (https://github.com/SwissTPH/malclimsim) to be installed, which is also described in the above paper.
